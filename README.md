#IoT Weather Station#
##Barometer##
for measuring device of barometric pressure and temperature.  
using Arduino (or Raspberry Pi) and Adafruit MPL3115A2 sensor.  
the MPL3115A2 is able to  measure barometric puresssure and temperature.
